Ansible Controller Prerequisite

The Controller must trust the **AD Root CA**. Without this certificate (Path shown in Gray), Ansible cannot validate the SSL identity of the 1,000+ hosts, breaking the Zero-Trust mandate.

Windows Node Hardening

**AD/CS** pushes GPO templates to ensure WinRM only listens on **5986 (HTTPS)**. It manages the lifecycle of the SSL certificates used for the secure handshake with Ansible.
